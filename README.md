# PokemonCards
Website to exchange pokemon cards you don't have with other people around the world.
## About
I developed this website in **php**, it's purpose is to be able to exchange pokemon cards that we already have for others we don't have.
## Functionalities
<ul>
  <li>Register Accounts server-side authentication</li>
  <li>Users can Login and Logout</li>
  <li>Modify User Password and Profile Picture</li>
  <li>Error Validation in every forms</li>
  <li>Table with names, ids, categories, amounts, of all pokemon cards</li>
  <li>the table can be sorted by all categories alphabetically</li>
  <li>Add and Remove any Pokemon in the desired amount</li>
  <li>User can use a contact form to get in touch with the administration</li>
  <li>Be able to evaluate all users and show the total votes, the average of the votes and your vote, in the user profile</li>
  <li>All pokemons have their own page with their characteristics</li>
  <li>Table with which people have that pokemon and how many</li>
  <li>In all profiles, there's a table with the pokemons they have and the ones they need</li>
  <li>At the top there is a search bar, which searches for users and pokemons, according to the search</li>
  <li>In the Admin area there's 2 types of it, admins and owners</li>
  <li>Admins have permission to set other users to admins or remove to normal user</li>
  <li>Admins can delete ssers profiles</li>
  <li>Admins can change passwords of users</li>
  <li>Admins can read the comments that regular users send</li>
  <li>The owner can do everything admins do, but no one can remove it from owner</li>
</ul>
